== Description ==

Minimalsm is a simple and clean Wordpress theme. However, this theme has everything you need: Gutenberg ready, SEO optimization, additional customization, translations and responsivity. Minimalism loads quickly and display any content nicely. It's totally free, so you can give it a try right now.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Minimalsm in the search form and press the 'Enter' key on your keyboard.
3. Find theme named 'Minimalsm' and click button 'Install' appeared on it.
4. After installing click button 'Activate'.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.

== Theme Limitations ==

Minimalsm theme does not display a 2nd or 3rd level child of the top (main) menu.
However, you can use a 2nd or 3rd level child menu on the second menu. This limitation is
justified for visual design purposes, the author designed the particular feature only for
second menu.
